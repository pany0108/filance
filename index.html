What is FILANCE?

FILANCE is an inter-blockchain data access system
that provides gateways for legacy blockchain
platforms to use decentralized storage,
while managing interfaces independently and securely.

FILANCE System Overview

Advantages

Automated Storage Management

Automates the process of leasing, updating, deleting, and restoring decentralized storag.

Secure and Robust Data Encryption

Increases usability with a convenient secret key management system, automated data encryption/decryption, and self error correction encoding.

High Availability
and Stability

Provides some of its own storage power directly to users for stable interlock with decentralized storage and user’s distributed repository.
